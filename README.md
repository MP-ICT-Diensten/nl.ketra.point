This app will integrate the minute point into the Athom homey

There will be 2 devices added:

####* Minut Point:
  * Read Data for the following items:
    * Pressure
    * Temperature
    * humidity
    * sound.
    * light
  * React to alarms of:
    * High/Low pressure.
    * High/low temperature.
    * High/low humidity.
    * High/low sound.
    * Noise
    * Glassbreak
    * Motion.
    
####* PointHome:
  * Read Data for:
    * Alarm State.
  * React to:
    * change of alarm state
